Statistical Analysis

Simple Linear Regression

Cancer Mortalities and Poverty Rates - via JMP, R
 An examination of a possible connection between poverty rates across America and correlated cancer mortality rates. I discuss the validity of the simple linear regression results, as well as look into any outlier cases, coming to the conclusion that poverty rates accounted for about 18% of the variability in cancer mortalities across the United States, based on the sample used.

Multiple Linear Regression

North Carolina Birth Weights - via JMP, R
 A regression using multiple variables to predict birth weights of North Carolinian babies, including the mother's minority status, whether she smokes, how many children she had previously birthed, etc. In combination, these variables explained about 46% of differences seen in the birth weights of babies.

Vizualizations

Minnesota Early/Absentee Voting

Hosted on Tableau Public - via Tableau, Excel
 A look into the increased popularity of early and absentee voting for presidential elections in Minnesota, especially as it pertained to the 2020 election. Marked by the COVID-19 pandemic, in-person voting was advised against and the option of voting absentee was greatly encouraged. These visualizations include a look over the past 2 decades of elections, a breakdown by county for the 2016 and 2020 elections, and an infographic showing some demographics of early voting.

- via Python

Original messy text file
 This task was to take this original messy text file and clean it using Python. First, unwanted lines such as blank lines, page number lines, and repeated headers were removed by matching regex patterns. Next, as data for a single trade was spread across 2-4 lines (as opposed to only a single line), I grouped them together into single lines per trade. Then the next step was to split up the data into columns, done via regex, and fix some data errors and finally, reorder the columns to match the header.
